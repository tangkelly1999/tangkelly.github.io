DRIVER DISPLAY BOARD

The Midnight Sun Solar Car Team has been designing and building solar-powered electric vehicles at the University of Waterloo since 1988. With our next car, MS XIV, we’re creating a car that is both energy-efficient and practical. Midnight Sun proudly represents our university at both the American Solar Challenge and World Solar Challenge.

Goal of Driver Display System: We wanted to design a vehicle dashboard that acts as an instrument panel to the driver. It should show information such as speed, state of car, power usage percentage and charge on a LCD screen. The software will be run on the Raspberry Pi.

  • My task was to design a PCB that:
    • Handles CAN <-> UART translation between the CAN network and RasberryPi
    • Regulate voltage from the 12V supplied to 5V for the Pi
    • Automatically adjusts screen brightness by using an ambient light sensor
    • Contains the RTC for Pi
Skills developed: PCB design and layout using Altium Designer, board bring-up, JIRA, Git
